# nautilus scripts

Here are my scripts for `nautilus`.

Put these into `~/.local/share/nautilus/scripts/`.


```
$ cd ~/.local/share/nautilus/scripts
$ git clone http://osamuaoki.github.com/nautilus-scripts.git ./
```
